Cloud IPFS for Enterprises

E-IPFS is a hybrid IPFS service that embraces the advantages of decentralization and access security.

IPFS/Filecoin

OpenVPN

E-IPFS securely transmits and controls access through a private path through OpenVPN.
We aim for the latest updated open source project.

Solutions for the enterprise

E-IPFS provides customized services for businesses by convergence of IPFS/Filecoin and OpenVPN.
It is a hybrid service that has no storage space limitation and can be utilized inside the company.
